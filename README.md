> **A sipmle Python script for solving an API-based CTF challenge.**

---

## About

Automated interaction with different API endpoints, where tokens are obtained, verified for a key and lastly submitted together (token + key) for a FLAG. 
Tokens expire too quickly for the process to be done "manually". 

---

## Installation

1. **Clone the repo**:

    ```bash
    git clone https://github.com/NanoMan0/API_Muncher.git
    ```


2. **Install dependencies**:

    ```bash
    pip install -r requirements.txt
    ```

---

## Usage

1. Configure your API URL and endpoints in the script.
2. Run the script:

    ```bash
    python APIMuncher.py
    ```
